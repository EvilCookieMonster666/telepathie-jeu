# Télépathie

Prototype de jeu de transmissions à deux joueurs, en ligne, relié à Supabase.

Ouvrir `index.html` depuis un serveur web (GitHub Pages, par exemple), puis créer ou rejoindre une partie grâce à un code de six caractères.

Ce prototype utilise des règles Supabase ouvertes pour les essais. Elles doivent être sécurisées avant toute publication plus large.
